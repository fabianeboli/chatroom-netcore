<div align="center">
  <h1>Chat App</h1>
</div>

An app in which user can:
- Create an account.
- Sign in.
- Create a new chatroom with an assigned name.
- Edit name of a chatroom.
- Delete chatroom created by a user.
- Join existing chatrooms.
- Write messages that will be seen by other users.
